Если название поля хранится в переменной, вы можете использовать метод getattr() для доступа к полю записи. Метод getattr() возвращает значение атрибута объекта по его имени.

Вот пример использования getattr() для доступа к полю записи по имени, хранящемуся в переменной:


field_name = 'my_field'  # Имя поля хранится в переменной
field_value = getattr(record, field_name)  # Получение значения поля из записи

print(field_value)  # Вывод значения поля
В этом примере record представляет собой объект записи, а field_name - переменную с именем поля. getattr(record, field_name) возвращает значение поля my_field из объекта record.

Обратите внимание, что если поле с указанным именем не существует в объекте записи, будет возбуждено исключение AttributeError. Поэтому убедитесь, что имя поля корректно передается в переменную field_name.
