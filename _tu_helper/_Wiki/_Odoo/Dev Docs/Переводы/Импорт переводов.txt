Для задания перевода для всех модулей в Odoo вы можете использовать модуль base_language_install. Этот модуль позволяет установить язык и импортировать переводы для всех установленных модулей одновременно.

Вот шаги, которые нужно выполнить для задания перевода для всех модулей:

Убедитесь, что модуль base_language_install установлен в вашей системе. Если его нет, установите его с помощью команды pip install odoo.addons.base_language_install.

Зайдите в интерфейс администратора Odoo и перейдите в раздел "Настройки".

В разделе "Пользователи" выберите "Пользователи" и найдите пользователя с ролью "Администратор". Если у вас нет пользователя с такой ролью, создайте нового пользователя и назначьте ему роль "Администратор".

Убедитесь, что у пользователя в поле "Язык" выбран нужный язык, на который вы хотите перевести модули.

Зайдите в раздел "Приложения" и выберите "Установленные приложения". Найдите модуль "Языковая установка" и установите его, если он не установлен.

После установки модуля "Языковая установка" вы увидите кнопку "Установить переводы". Нажмите на нее, чтобы начать процесс импорта переводов для всех модулей.

После завершения импорта переводов для всех модулей, вы сможете видеть переведенные строки на выбранном языке.

Обратите внимание, что для успешного импорта переводов, необходимо наличие доступа к интернету, так как переводы будут загружены с сервера Odoo.
