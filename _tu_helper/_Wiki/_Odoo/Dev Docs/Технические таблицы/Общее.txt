Во время установки модуля в Odoo заполняются несколько технических таблиц в базе данных. Некоторые из наиболее значимых таблиц, которые могут быть заполнены при установке модуля, включают:

ir_module_module: Эта таблица содержит информацию о всех установленных модулях в системе. При установке нового модуля добавляется запись в эту таблицу.

ir_model: В этой таблице хранится информация о моделях данных, определенных в модулях. При установке модуля добавляются новые записи для моделей, определенных в этом модуле.

ir_model_fields: Эта таблица содержит информацию о полях моделей данных. При установке модуля добавляются новые записи для полей, определенных в модуле.

ir_model_constraints: В этой таблице хранятся ограничения моделей данных. При установке модуля могут добавляться новые записи для ограничений, определенных в модуле.

ir_ui_view: Эта таблица содержит информацию о представлениях данных (view) моделей. При установке модуля добавляются новые записи для представлений, определенных в модуле.

ir_actions: В этой таблице хранятся информация о действиях (actions), таких как действия при нажатии на кнопки или пункты меню. При установке модуля могут добавляться новые записи для действий, определенных в модуле.

ir_model_relation: В этой таблице хранятся связи между моделями данных. При установке модуля могут добавляться новые записи для связей, определенных в модуле.

ir_ui_menu: В этой таблице хранится информация о структуре меню. При установке модуля могут добавляться новые записи для пунктов меню, определенных в модуле.

ir_ui_view_custom: Эта таблица содержит информацию о пользовательских изменениях представлений (view). При установке модуля могут добавляться новые записи для представлений, требующих пользовательских изменений.

ir_ui_view_group_rel: В этой таблице хранится информация о связях между представлениями и группами пользователей. При установке модуля могут добавляться новые записи для определения доступа к представлениям для определенных групп пользователей.

ir_module_category: В этой таблице хранится информация о категориях модулей. При установке модуля могут добавляться новые записи для категорий, определенных в модуле.

ir_model_access: В этой таблице хранится информация о правах доступа к моделям данных. При установке модуля могут добавляться новые записи для определения прав доступа для моделей, определенных в модуле.

ir_model_data: В этой таблице хранятся данные моделей, такие как идентификаторы записей и другая связанная информация. При установке модуля могут добавляться новые записи для данных, определенных в модуле.

ir_attachment: Эта таблица содержит информацию о прикрепленных файлах. При установке модуля могут добавляться новые записи для файлов, связанных с модулем.

ir_model_relation_reference: В этой таблице хранится информация о ссылках на связи между моделями данных. При установке модуля могут добавляться новые записи для определения ссылок на связи.

ir_model_data_char_index: Эта таблица используется для поиска записей в таблице ir_model_data по значению поля name.

ir_actions: В этой таблице хранится информация об акциях (действиях), таких как создание, изменение, удаление записей. При установке модуля могут добавляться новые записи для определения новых акций.

ir_actions_act_window: Эта таблица содержит информацию о действиях типа "act_window" (окно). При установке модуля могут добавляться новые записи для определения новых действий типа "act_window".

ir_actions_server: В этой таблице хранится информация о серверных действиях, которые выполняются на стороне сервера. При установке модуля могут добавляться новые записи для определения новых серверных действий.

ir_rule: Эта таблица содержит информацию о правилах доступа, определенных для моделей данных. При установке модуля могут добавляться новые записи для определения новых правил доступа.

ir_filters: В этой таблице хранится информация о фильтрах, которые могут быть применены к представлениям данных. При установке модуля могут добавляться новые записи для определения новых фильтров.

ir_sequence: последовательности (номерах документов).
При установке модуля могут добавляться новые записи для определения новых последовательностей.

ir_default: Эта таблица содержит информацию о значениях по умолчанию для полей моделей данных. При установке модуля могут добавляться новые записи для определения новых значений по умолчанию.
