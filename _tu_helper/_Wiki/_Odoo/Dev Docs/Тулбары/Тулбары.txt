тулбары:

Тулбар приложения - содержит список всех установленных приложений и позволяет быстро переключаться между ними.
Тулбар представления - содержит кнопки для фильтрации, группировки, сортировки и переключения между различными представлениями данных, такими как список, календарь и график.
Тулбар формы - содержит кнопки для сохранения, отмены и удаления изменений в форме, а также для выполнения различных операций, таких как отправка сообщения, создание связанных записей и т.д.
Тулбар Действий - содержит кнопки для выполнения различных операций над выбранными записями, таких как удаление, экспорт, печать, выполнение действий сопоставленных с моделью (таких как "Выставить счет" или "Отправить письмо").
Тулбар поиска - содержит поле для поиска записей по ключевым словам и фильтрует результаты в соответствии с введенным запросом.
Тулбар боковой панели - отображается на боковой панели и содержит кнопки для управления видимостью боковой панели и ее содержимого.
