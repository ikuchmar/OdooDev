Сравнения двух словарей и указания ключей, у которых в списках зависимостей разные значения:


depends_1 = {
    'm5': ['m1'],
    'm2': ['m3', 'm5'],
    'm4': ['m3'],
    'm1': ['m2', 'm3'],
    'm3': [],
}

depends_2 = {
    'm3': [],
    'm4': ['m3'],
    'm2': ['m3'],
    'm1': ['m2', 'm3'],
    'm5': ['m1'],
}

# Создаем список ключей, объединяя ключи обоих словарей
keys = set(list(depends_1.keys()) + list(depends_2.keys()))

# Создаем пустой словарь для различающихся значений
different_values = {}

# Проходим по каждому ключу
for key in keys:
    # Получаем значения из обоих словарей
    values_1 = depends_1.get(key, [])
    values_2 = depends_2.get(key, [])

    # Сравниваем значения
    if values_1 != values_2:
        different_values[key] = (values_1, values_2)

# Выводим различающиеся значения
for key, values in different_values.items():
    print(f"Key: {key}, Values_1: {values[0]}, Values_2: {values[1]}")
Результат выполнения скрипта будет:



Key: m2, Values_1: ['m3', 'm5'], Values_2: ['m3']
Key: m5, Values_1: ['m1'], Values_2: ['m1']
Скрипт сравнивает значения для каждого ключа из обоих словарей и выводит ключи, у которых значения отличаются. В данном случае, значения для ключей 'm2' и 'm5' различаются.
