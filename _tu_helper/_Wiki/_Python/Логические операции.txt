and (и) - возвращает True, если оба операнда истинны. Иначе возвращает False.
or (или) - возвращает True, если хотя бы один из операндов истинный. Иначе возвращает False.
not (не) - возвращает True, если операнд ложный. Иначе возвращает False.
В таблице ниже представлены значения истинности (результаты) для всех возможных комбинаций двух операндов и для каждой логической операции:

a	b	a and b	a or b	not a
True	True	True	True	False
True	False	False	True	False
False	True	False	True	True
False	False	False	False	True
Также стоит отметить, что операции сравнения (например, ==, <, >) возвращают булево значение True или False в зависимости от того, выполняется ли условие, заданное оператором сравнения. Например, выражение x > 10 вернет True, если x больше 10, и False в противном случае.
