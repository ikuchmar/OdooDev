## array-methods
categories: Web - JavaScript - Массивы
aliases: map, filter, reduce
Частые методы массивов: map, filter, reduce.
```javascript
[1,2,3].map(x=>x*2)
```
```javascript
[1,2,3].filter(x=>x%2===0)
```
