Горячие клавиши:
===========================

Ctrl + / комментирования выделенного куска кода
Ctrl + Alt + L Форматирование кода

Tab Сдвиг вправо (увеличить отступ)
Shift + Tab Сдвиг влево (уменьшить отступ)
Alt + Shift + ↑ / ↓  Вверх/вниз	


найти файл по имени:
===========================

Alt + F7 - поиск во всех - в ввиде комбобокса
Ctrl + Alt + F - поисх по всех - в виде отдной обработки

Использование поиска по файлам (Find in Files): Вы можете воспользоваться функцией поиска по файлам, чтобы найти файлы
по имени. Для этого вы можете выполнить следующие шаги:
a. Выберите в меню "Edit" (Правка) -> "Find" (Найти) -> "Find in Files" (Найти в файлах), или нажмите сочетание клавиш
Ctrl+Shift+F (Windows/Linux) или Command+Shift+F (Mac).
b. В появившемся окне "Find in Files" (Найти в файлах) введите имя файла или паттерн имени файла в поле "File name
mask" (Маска имени файла), выберите директорию, в которой хотите выполнить поиск, и нажмите на кнопку "Find" (Найти).

Использование быстрого поиска (Navigate to File): PyCharm также предлагает функцию быстрого поиска файлов по имени. Для
этого можно выполнить следующие действия:
a. Нажмите два раза на клавишу Shift, чтобы открыть окно быстрого поиска.
b. Введите имя файла или паттерн имени файла в поле поиска.
c. Выберите файл, который вы ищете, из списка предложенных результатов и нажмите Enter, чтобы открыть его.

Использование окна "Project" (Проект): Если вы хотите найти файл по имени в контексте вашего проекта, вы также можете
воспользоваться окном "Project" (Проект), которое отображает структуру вашего проекта. Для этого можно выполнить
следующие действия:
a. Откройте окно "Project" (Проект), нажав на вкладку "Project" (Проект) в левой части PyCharm.
b. Введите имя файла или паттерн имени файла в поле "Name" (Имя) в верхней части окна "Project" (Проект). PyCharm
автоматически фильтрует файлы по имени, отображая только те, которые соответствуют вашему запросу.
c. Выберите файл, который вы ищете, из списка файлов в окне "Project" (Проект), и дважды щелкните на нем, чтобы открыть
его.
