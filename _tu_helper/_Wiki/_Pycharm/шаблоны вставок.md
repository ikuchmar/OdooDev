Основные варианты:
Live Templates: быстрые вставки по аббревиатуре.
Откройте File \| Settings \| Editor \| Live Templates.
Выберите группу или создайте свою, нажмите + → Live Template.
Введите текст шаблона, задайте Abbreviation и Description.
Укажите Applicable contexts (где шаблон доступен).
Вызов: печатайте аббревиатуру и нажмите Tab (или настроенную клавишу).

Postfix Templates: расширяют выражения по суффиксу (например, .if, .for).
В File \| Settings \| Editor \| Postfix Completion включите/создайте свои.

File Templates: шаблоны для новых файлов.
В File \| Settings \| Editor \| File and Code Templates.
Custom Snippets через Live Templates для текста в маркдауне, Python, и т.\ п.
Горячие клавиши:
Вызов заполнителей: Tab после аббревиатуры.
Расширение шаблона принудительно: Ctrl+J (вставка Live Template из списка).
Настройка клавиш: File \| Settings \| Keymap → найдите Insert Live Template и назначьте хоткей.
Для Python:
Добавьте контекст Python в шаблоне, чтобы он работал в .py файлах.
Поля вида $VAR$ можно заполнять автоматически и навигировать по ним Tab.
