
# –Æ–Ω–∏–∫–æ–¥, –≤–Ω—É—Ç—Ä–µ–Ω–Ω–µ–µ —É—Å—Ç—Ä–æ–π—Å—Ç–≤–æ —Å—Ç—Ä–æ–∫


–ö–∞–∫ –º—ã —É–∂–µ –∑–Ω–∞–µ–º, —Å—Ç—Ä–æ–∫–∏ –≤ JavaScript –æ—Å–Ω–æ–≤–∞–Ω—ã –Ω–∞ [–Æ–Ω–∏–∫–æ–¥–µ](https://ru.wikipedia.org/wiki/–Æ–Ω–∏–∫–æ–¥): –∫–∞–∂–¥—ã–π —Å–∏–º–≤–æ–ª –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç –∏–∑ —Å–µ–±—è –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å –±–∞–π—Ç–æ–≤ –∏–∑ 1-4 –±–∞–π—Ç–æ–≤.

JavaScript –ø–æ–∑–≤–æ–ª—è–µ—Ç –Ω–∞–º –≤—Å—Ç–∞–≤–∏—Ç—å —Å–∏–º–≤–æ–ª –≤ —Å—Ç—Ä–æ–∫—É, —É–∫–∞–∑–∞–≤ –µ–≥–æ —à–µ—Å—Ç–Ω–∞–¥—Ü–∞—Ç–µ—Ä–∏—á–Ω—ã–π –Æ–Ω–∏–∫–æ–¥ —Å –ø–æ–º–æ—â—å—é –æ–¥–Ω–æ–π –∏–∑ —ç—Ç–∏—Ö —Ç—Ä–µ—Ö –Ω–æ—Ç–∞—Ü–∏–π:

- `\xXX`

    –í–º–µ—Å—Ç–æ `XX` –¥–æ–ª–∂–Ω—ã –±—ã—Ç—å —É–∫–∞–∑–∞–Ω—ã –¥–≤–µ —à–µ—Å—Ç–Ω–∞–¥—Ü–∞—Ç–µ—Ä–∏—á–Ω—ã–µ —Ü–∏—Ñ—Ä—ã —Å–æ –∑–Ω–∞—á–µ–Ω–∏–µ–º –æ—Ç `00` –¥–æ `FF`. –í —ç—Ç–æ–º —Å–ª—É—á–∞–µ `\xXX` -- —ç—Ç–æ —Å–∏–º–≤–æ–ª, –Æ–Ω–∏–∫–æ–¥ –∫–æ—Ç–æ—Ä–æ–≥–æ —Ä–∞–≤–µ–Ω `XX`.

    –ü–æ—Å–∫–æ–ª—å–∫—É –Ω–æ—Ç–∞—Ü–∏—è `\xXX` –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞–µ—Ç —Ç–æ–ª—å–∫–æ –¥–≤–µ —à–µ—Å—Ç–Ω–∞–¥—Ü–∞—Ç–µ—Ä–∏—á–Ω—ã–µ —Ü–∏—Ñ—Ä—ã, –µ–µ –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Ç–æ–ª—å–∫–æ –¥–ª—è –ø–µ—Ä–≤—ã—Ö 256 —Å–∏–º–≤–æ–ª–æ–≤ –Æ–Ω–∏–∫–æ–¥–∞.

    –≠—Ç–∏ 256 —Å–∏–º–≤–æ–ª–æ–≤ –≤–∫–ª—é—á–∞—é—Ç –≤ —Å–µ–±—è –ª–∞—Ç–∏–Ω—Å–∫–∏–π –∞–ª—Ñ–∞–≤–∏—Ç, –±–æ–ª—å—à–∏–Ω—Å—Ç–≤–æ –æ—Å–Ω–æ–≤–Ω—ã—Ö —Å–∏–Ω—Ç–∞–∫—Å–∏—á–µ—Å–∫–∏—Ö —Å–∏–º–≤–æ–ª–æ–≤ –∏ –Ω–µ–∫–æ—Ç–æ—Ä—ã–µ –¥—Ä—É–≥–∏–µ. –ù–∞–ø—Ä–∏–º–µ—Ä, `"\x7A"` - —ç—Ç–æ —Ç–æ –∂–µ —Å–∞–º–æ–µ, —á—Ç–æ `"z"` (–Æ–Ω–∏–∫–æ–¥ `U+007A`).

    ```js run

    alert( "\xA9" ); // ¬©, —Å–∏–º–≤–æ–ª –∞–≤—Ç–æ—Ä—Å–∫–æ–≥–æ –ø—Ä–∞–≤–∞
    ```

- `\uXXXX`
    
    –í–º–µ—Å—Ç–æ `XXXX` –¥–æ–ª–∂–Ω—ã –±—ã—Ç—å —É–∫–∞–∑–∞–Ω—ã —Ä–æ–≤–Ω–æ 4 —à–µ—Å—Ç–Ω–∞–¥—Ü–∞—Ç–µ—Ä–∏—á–Ω—ã–µ —Ü–∏—Ñ—Ä—ã —Å–æ –∑–Ω–∞—á–µ–Ω–∏–µ–º –æ—Ç `0000` –¥–æ `FFFF`. –í —ç—Ç–æ–º —Å–ª—É—á–∞–µ `\uXXXX` - —ç—Ç–æ —Å–∏–º–≤–æ–ª, –Æ–Ω–∏–∫–æ–¥ –∫–æ—Ç–æ—Ä–æ–≥–æ —Ä–∞–≤–µ–Ω `XXXX`.

    –°–∏–º–≤–æ–ª—ã —Å–æ –∑–Ω–∞—á–µ–Ω–∏—è–º–∏ –Æ–Ω–∏–∫–æ–¥–∞, –ø—Ä–µ–≤—ã—à–∞—é—â–∏–º–∏ `U+FFFF`, —Ç–∞–∫–∂–µ –º–æ–≥—É—Ç –±—ã—Ç—å –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω—ã —Å –ø–æ–º–æ—â—å—é —ç—Ç–æ–π –Ω–æ—Ç–∞—Ü–∏–∏, –Ω–æ –≤ —Ç–∞–∫–æ–º —Å–ª—É—á–∞–µ –Ω–∞–º –ø—Ä–∏–¥–µ—Ç—Å—è –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Ç–∞–∫ –Ω–∞–∑—ã–≤–∞–µ–º—É—é —Å—É—Ä—Ä–æ–≥–∞—Ç–Ω—É—é –ø–∞—Ä—É (–æ –Ω–µ–π –º—ã –ø–æ–≥–æ–≤–æ—Ä–∏–º –ø–æ–∑–∂–µ –≤ —ç—Ç–æ–π –≥–ª–∞–≤–µ).
    
    ```js run
    alert( "\u00A9" ); // ¬©, —Ç–æ –∂–µ —Å–∞–º–æ–µ, —á—Ç–æ \xA9, –∏—Å–ø–æ–ª—å–∑—É—è 4-–∑–Ω–∞—á–Ω—É—é —à–µ—Å—Ç–Ω–∞–¥—Ü–∞—Ç–µ—Ä–∏—á–Ω—É—é –Ω–æ—Ç–∞—Ü–∏—é
    alert( "\u044F" ); // —è, –±—É–∫–≤–∞ –∫–∏—Ä–∏–ª–ª–∏—á–µ—Å–∫–æ–≥–æ –∞–ª—Ñ–∞–≤–∏—Ç–∞
    alert( "\u2191" ); // ‚Üë, —Å–∏–º–≤–æ–ª —Å—Ç—Ä–µ–ª–∫–∏ –≤–≤–µ—Ä—Ö
    ```

- `\u{X‚Ä¶XXXXXX}`

    –í–º–µ—Å—Ç–æ `X‚Ä¶XXXXXX` –¥–æ–ª–∂–Ω–æ –±—ã—Ç—å —à–µ—Å—Ç–Ω–∞–¥—Ü–∞—Ç–µ—Ä–∏—á–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –æ—Ç 1 –¥–æ 6 –±–∞–π—Ç –æ—Ç `0` –¥–æ `10FFFF` (–º–∞–∫—Å–∏–º–∞–ª—å–Ω–∞—è —Ç–æ—á–∫–∞ –∫–æ–¥–∞, –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω–∞—è —Å—Ç–∞–Ω–¥–∞—Ä—Ç–æ–º –Æ–Ω–∏–∫–æ–¥). –≠—Ç–∞ –Ω–æ—Ç–∞—Ü–∏—è –ø–æ–∑–≤–æ–ª—è–µ—Ç –Ω–∞–º –ª–µ–≥–∫–æ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—Ç—å –≤—Å–µ —Å—É—â–µ—Å—Ç–≤—É—é—â–∏–µ —Å–∏–º–≤–æ–ª—ã –Æ–Ω–∏–∫–æ–¥–∞.

## –°—É—Ä—Ä–æ–≥–∞—Ç–Ω—ã–µ –ø–∞—Ä—ã

–í—Å–µ —á–∞—Å—Ç–æ –∏—Å–ø–æ–ª—å–∑—É–µ–º—ã–µ —Å–∏–º–≤–æ–ª—ã –∏–º–µ—é—Ç 2-–±–∞–π—Ç–æ–≤—ã–µ –∫–æ–¥—ã (4 —à–µ—Å—Ç–Ω–∞–¥—Ü–∞—Ç–µ—Ä–∏—á–Ω—ã–µ —Ü–∏—Ñ—Ä—ã). –í –±–æ–ª—å—à–∏–Ω—Å—Ç–≤–µ –µ–≤—Ä–æ–ø–µ–π—Å–∫–∏—Ö —è–∑—ã–∫–æ–≤ –±—É–∫–≤—ã, —Ü–∏—Ñ—Ä—ã –∏ –æ—Å–Ω–æ–≤–Ω—ã–µ —É–Ω–∏—Ñ–∏—Ü–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ –∏–¥–µ–æ–≥—Ä–∞—Ñ–∏—á–µ—Å–∫–∏–µ –Ω–∞–±–æ—Ä—ã CJK (CJK -- –æ—Ç –∫–∏—Ç–∞–π—Å–∫–æ–π, —è–ø–æ–Ω—Å–∫–æ–π –∏ –∫–æ—Ä–µ–π—Å–∫–æ–π —Å–∏—Å—Ç–µ–º –ø–∏—Å—å–º–∞) –∏–º–µ—é—Ç 2-–±–∞–π—Ç–æ–≤–æ–µ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏–µ.

–ò–∑–Ω–∞—á–∞–ª—å–Ω–æ JavaScript –±—ã–ª –æ—Å–Ω–æ–≤–∞–Ω –Ω–∞ –∫–æ–¥–∏—Ä–æ–≤–∫–µ UTF-16, –∫–æ—Ç–æ—Ä–∞—è –ø—Ä–µ–¥—É—Å–º–∞—Ç—Ä–∏–≤–∞–ª–∞ —Ç–æ–ª—å–∫–æ 2 –±–∞–π—Ç–∞ –Ω–∞ –æ–¥–∏–Ω —Å–∏–º–≤–æ–ª. –û–¥–Ω–∞–∫–æ 2 –±–∞–π—Ç–∞ –¥–æ–ø—É—Å–∫–∞—é—Ç —Ç–æ–ª—å–∫–æ 65536 –∫–æ–º–±–∏–Ω–∞—Ü–∏–π, –∏ —ç—Ç–æ–≥–æ –Ω–µ–¥–æ—Å—Ç–∞—Ç–æ—á–Ω–æ –¥–ª—è –≤—Å–µ—Ö –≤–æ–∑–º–æ–∂–Ω—ã—Ö —Å–∏–º–≤–æ–ª–æ–≤ –Æ–Ω–∏–∫–æ–¥–∞.

–ü–æ—ç—Ç–æ–º—É —Ä–µ–¥–∫–∏–µ —Å–∏–º–≤–æ–ª—ã, —Ç—Ä–µ–±—É—é—â–∏–µ –±–æ–ª–µ–µ 2 –±–∞–π—Ç, –∫–æ–¥–∏—Ä—É—é—Ç—Å—è –ø–∞—Ä–æ–π 2-–±–∞–π—Ç–æ–≤—ã—Ö —Å–∏–º–≤–æ–ª–æ–≤, –∫–æ—Ç–æ—Ä—ã–µ –Ω–∞–∑—ã–≤–∞—é—Ç—Å—è "—Å—É—Ä—Ä–æ–≥–∞—Ç–Ω–æ–π –ø–∞—Ä–æ–π".

–ü–æ–±–æ—á–Ω—ã–º —ç—Ñ—Ñ–µ–∫—Ç–æ–º —è–≤–ª—è–µ—Ç—Å—è —Ç–æ, —á—Ç–æ –¥–ª–∏–Ω–∞ —Ç–∞–∫–∏—Ö —Å–∏–º–≤–æ–ª–æ–≤ —Ä–∞–≤–Ω–∞ `2`:

```js run
alert( 'ùí≥'.length ); // 2, MATHEMATICAL SCRIPT CAPITAL X
alert( 'üòÇ'.length ); // 2, FACE WITH TEARS OF JOY
alert( '©∑∂'.length ); // 2, —Ä–µ–¥–∫–∏–π –∫–∏—Ç–∞–π—Å–∫–∏–π –∏–µ—Ä–æ–≥–ª–∏—Ñ
```

–≠—Ç–æ –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç –ø–æ—Ç–æ–º—É, —á—Ç–æ —Å—É—Ä—Ä–æ–≥–∞—Ç–Ω—ã–µ –ø–∞—Ä—ã –Ω–µ —Å—É—â–µ—Å—Ç–≤–æ–≤–∞–ª–∏ –≤ —Ç–æ –≤—Ä–µ–º—è, –∫–æ–≥–¥–∞ –±—ã–ª —Å–æ–∑–¥–∞–Ω JavaScript, –∏ –ø–æ—ç—Ç–æ–º—É –æ–Ω–∏ –Ω–µ –æ–±—Ä–∞–±–∞—Ç—ã–≤–∞—é—Ç—Å—è —è–∑—ã–∫–æ–º –∫–æ—Ä—Ä–µ–∫—Ç–Ω–æ.

–ù–∞ —Å–∞–º–æ–º –¥–µ–ª–µ –≤ –∫–∞–∂–¥–æ–π –∏–∑ –ø—Ä–∏–≤–µ–¥–µ–Ω–Ω—ã—Ö —Å—Ç—Ä–æ–∫ —É –Ω–∞—Å –ø–æ –æ–¥–Ω–æ–º—É —Å–∏–º–≤–æ–ª—É, –Ω–æ —Å–≤–æ–π—Å—Ç–≤–æ `length` –ø–æ–∫–∞–∑—ã–≤–∞–µ—Ç –¥–ª–∏–Ω—É `2`.

–ü–æ–ª—É—á–∏—Ç—å —Ç–∞–∫–æ–π —Å–∏–º–≤–æ–ª —Ç–∞–∫–∂–µ –º–æ–∂–µ—Ç –±—ã—Ç—å –Ω–µ–ø—Ä–æ—Å—Ç–æ, –ø–æ—Å–∫–æ–ª—å–∫—É –±–æ–ª—å—à–∏–Ω—Å—Ç–≤–æ —è–∑—ã–∫–æ–≤—ã—Ö —Ñ—É–Ω–∫—Ü–∏–π —Ä–∞—Å—Å–º–∞—Ç—Ä–∏–≤–∞—é—Ç —Å—É—Ä—Ä–æ–≥–∞—Ç–Ω—ã–µ –ø–∞—Ä—ã –∫–∞–∫ –¥–≤–∞ —Å–∏–º–≤–æ–ª–∞.

–ù–∞–ø—Ä–∏–º–µ—Ä, –∑–¥–µ—Å—å –º—ã –≤–∏–¥–∏–º –¥–≤–∞ —Å—Ç—Ä–∞–Ω–Ω—ã—Ö —Å–∏–º–≤–æ–ª–∞ –≤ –≤—ã–≤–æ–¥–µ:

```js run

alert( 'ùí≥'[1] ); // ...—á–∞—Å—Ç–∏ —Å—É—Ä—Ä–æ–≥–∞—Ç–Ω–æ–π –ø–∞—Ä—ã
```

–ß–∞—Å—Ç–∏ —Å—É—Ä—Ä–æ–≥–∞—Ç–Ω–æ–π –ø–∞—Ä—ã –Ω–µ –∏–º–µ—é—Ç –Ω–∏–∫–∞–∫–æ–≥–æ –∑–Ω–∞—á–µ–Ω–∏—è –¥—Ä—É–≥ –±–µ–∑ –¥—Ä—É–≥–∞.

–¢–µ—Ö–Ω–∏—á–µ—Å–∫–∏, —Å—É—Ä—Ä–æ–≥–∞—Ç–Ω—ã–µ –ø–∞—Ä—ã —Ç–∞–∫–∂–µ –º–æ–∂–Ω–æ –æ–ø—Ä–µ–¥–µ–ª–∏—Ç—å –ø–æ –∏—Ö –∫–æ–¥–∞–º: –µ—Å–ª–∏ —Å–∏–º–≤–æ–ª –∏–º–µ–µ—Ç –∫–æ–¥ –≤ –∏–Ω—Ç–µ—Ä–≤–∞–ª–µ `0xd800...0xdbff`, —Ç–æ –æ–Ω —è–≤–ª—è–µ—Ç—Å—è –ø–µ—Ä–≤–æ–π —á–∞—Å—Ç—å—é —Å—É—Ä—Ä–æ–≥–∞—Ç–Ω–æ–π –ø–∞—Ä—ã. –°–ª–µ–¥—É—é—â–∏–π —Å–∏–º–≤–æ–ª (–≤—Ç–æ—Ä–∞—è —á–∞—Å—Ç—å) –¥–æ–ª–∂–µ–Ω –∏–º–µ—Ç—å –∫–æ–¥ –≤ –∏–Ω—Ç–µ—Ä–≤–∞–ª–µ `0xdc00...0xdfff`. –≠—Ç–∏ –∏–Ω—Ç–µ—Ä–≤–∞–ª—ã –∑–∞—Ä–µ–∑–µ—Ä–≤–∏—Ä–æ–≤–∞–Ω—ã —Å—Ç–∞–Ω–¥–∞—Ä—Ç–æ–º –∏—Å–∫–ª—é—á–∏—Ç–µ–ª—å–Ω–æ –¥–ª—è —Å—É—Ä—Ä–æ–≥–∞—Ç–Ω—ã—Ö –ø–∞—Ä.

–ü–æ—ç—Ç–æ–º—É –¥–ª—è —Ä–∞–±–æ—Ç—ã —Å —Å—É—Ä—Ä–æ–≥–∞—Ç–Ω—ã–º–∏ –ø–∞—Ä–∞–º–∏ –≤ JavaScript –±—ã–ª–∏ –¥–æ–±–∞–≤–ª–µ–Ω—ã –º–µ—Ç–æ–¥—ã `String.fromCodePoint` –∏ `str.codePointAt`.

–ü–æ —Å—É—Ç–∏, –æ–Ω–∏ –∞–Ω–∞–ª–æ–≥–∏—á–Ω—ã [String.fromCharCode](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/String/fromCharCode) –∏ [str.charCodeAt](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/String/charCodeAt), –Ω–æ –æ–Ω–∏ –ø—Ä–∞–≤–∏–ª—å–Ω–æ –æ–±—Ä–∞–±–∞—Ç—ã–≤–∞—é—Ç —Å—É—Ä—Ä–æ–≥–∞—Ç–Ω—ã–µ –ø–∞—Ä—ã.

–ó–¥–µ—Å—å –º–æ–∂–Ω–æ —É–≤–∏–¥–µ—Ç—å —Ä–∞–∑–Ω–∏—Ü—É:

```js run
// charCodeAt –Ω–µ —É—á–∏—Ç—ã–≤–∞–µ—Ç —Å—É—Ä—Ä–æ–≥–∞—Ç–Ω—ã–µ –ø–∞—Ä—ã, –ø–æ—ç—Ç–æ–º—É –æ–Ω –≤—ã–¥–∞–µ—Ç –∫–æ–¥—ã –¥–ª—è 1-–π —á–∞—Å—Ç–∏ ùí≥:

```

–ü—Ä–∏ —ç—Ç–æ–º, –µ—Å–ª–∏ –±—Ä–∞—Ç—å —Å –ø–æ–∑–∏—Ü–∏–∏ 1 (–∞ —ç—Ç–æ –∑–¥–µ—Å—å —Å–∫–æ—Ä–µ–µ –Ω–µ–≤–µ—Ä–Ω–æ), —Ç–æ –æ–Ω–∏ –æ–±–∞ –≤–æ–∑–≤—Ä–∞—â–∞—é—Ç —Ç–æ–ª—å–∫–æ 2-—é —á–∞—Å—Ç—å –ø–∞—Ä—ã:
