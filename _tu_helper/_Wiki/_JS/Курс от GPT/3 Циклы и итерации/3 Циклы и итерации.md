3: Циклы и итерации
📌 Цель:
Научиться повторять действия с помощью циклов: for, while, for...of.

🧾 Теория:
🔸 Цикл for
javascript
Copy
Edit
for (let i = 0; i < 5; i++) {
    console.log("Шаг", i);
}
⏱ Что происходит:

i = 0 — начальное значение

i < 5 — условие выполнения

i++ — шаг цикла (увеличиваем на 1)

🔸 Цикл while
javascript
Copy
Edit
let i = 0;
while (i < 3) {
    console.log("Число", i);
    i++;
}
🔸 Цикл for...of — по массивам
javascript
Copy
Edit
let fruits = ["apple", "banana", "cherry"];
for (let fruit of fruits) {
    console.log(fruit);
}
🔸 break и continue
break — завершает весь цикл

continue — пропускает текущую итерацию

javascript
Copy
Edit
for (let i = 0; i < 5; i++) {
    if (i === 3) continue;
    console.log(i);
}
